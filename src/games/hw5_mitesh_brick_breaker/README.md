# hw5_mitesh_brick_breaker
It's ball vs. bricks - who will win? Keep the ball bouncing as you smash through layers of bricks!

## Controls
| Input        | Action              |
|--------------|---------------------|
| `LeftArrow`  | Move platform left  |
| `RightArrow` | Move platform right |
