# hw5_mitesh_bubble_shooter
Shoot bubbles and clear the board!

## Controls
| Input        | Action             |
|--------------|--------------------|
| `LeftArrow`  | Move shooter left  |
| `RightArrow` | Move shooter right |
| `Space`      | Shoot bubble       |

Common engine controls can be found in the engine [README](../../../README.md#common-inputs)
